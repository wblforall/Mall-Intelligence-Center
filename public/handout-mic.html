1. Gambaran Umum & Arsitektur

http://localhost/mall-intelligence-center/public/

Mall Intelligence Center (MIC) adalah sistem manajemen operasional terpadu untuk dua properti PT. Wulandari Bangun Laksana Tbk.: eWalk Simply FUNtastic dan Pentacity Shopping Venue. Sistem ini mencakup perencanaan dan evaluasi event, monitoring traffic pengunjung, analisis ekonomi, hingga pengembangan SDM.

🎪 Event Management

  • Perencanaan event dual-mall
  • Pengelolaan konten & rundown
  • Koordinasi exhibitor & sponsor
  • Visual merchandising
  • Loyalty program per event

📊 Analytics & Intelligence

  • Traffic harian & jam-an per pintu
  • KPI event (budget vs realisasi)
  • ROI & funnel conversion
  • Perbandingan antar event
  • Indikator ekonomi makro real-time

👥 People Development

  • Master data karyawan
  • Survey keterlibatan (EEI)
  • Training Needs Analysis (TNA)
  • Manajemen program pelatihan
  • Budget pelatihan per departemen

11. Traffic Pengunjung

/traffic
Semua Logistik / Operasional (Input) Admin (Import)

Sistem pencatatan dan analitik pengunjung harian per pintu masuk, tersedia untuk kedua mall secara terpisah.

✏️ Input Traffic Harian

  • Grid input: jam (10:00–23:00) × pintu mall
  • Isi jumlah pengunjung per sel; total per jam dihitung otomatis
  • Input kendaraan: total mobil & motor
  • Data tersimpan per tanggal per mall
  • Overwrite jika sudah ada data hari itu

📊 Traffic Summary

  • Pilih rentang tanggal (from–to)
  • KPI: total pengunjung, total mobil, total motor
  • Chart: tren harian eWalk vs Pentacity
  • Chart: tren kendaraan (mobil & motor)
  • Chart: distribusi jam-an per mall
  • Breakdown per pintu masuk
  • Insight: peak hour, best day, rata-rata harian

⚖️ Traffic Compare

  • Bandingkan dua periode berbeda
  • Chart daily: H1, H2, ... per periode (side by side)
  • Chart hourly per mall
  • % perubahan per periode
  • Best day comparison

📥 Import Excel (Bulk)

  • Upload 1 file → preview detail + konfirmasi tanggal manual
  • Upload banyak file → preview summary (file, tanggal, total, pintu)
  • Parser otomatis: deteksi tanggal, jam (baris) × pintu (kolom)
  • eWalk: baris UG Funstation & FF XXI dikenali khusus
  • Semua import dicatat di activity log
Relasi Traffic ↔ Event Traffic tidak disimpan per event — data tersimpan per tanggal. Saat menghitung traffic sebuah event, sistem mengambil data dari tanggal mulai hingga tanggal selesai event secara otomatis.

18. Employee Engagement Index (EEI)

/people/eei
HR Semua Karyawan (Survey)

Survey keterlibatan karyawan yang bersifat anonim, dikelola per periode, dengan hasil agregat per dimensi, departemen, dan level jabatan.

📊 Dashboard Hasil

  • Overall score EEI (0–100) dengan gauge visual
  • Skor per dimensi dengan progress bar warna
  • Skor per departemen (tabel + indikator Engaged/Moderate/Disengaged)
  • Skor per level jabatan
  • Matriks Departemen × Level Jabatan (cross-tab)
  • Tingkat partisipasi: selesai / total karyawan aktif (%)

📝 Survey Karyawan

  • Hanya bisa diisi saat periode aktif
  • Pertanyaan skala Likert 1–5 per dimensi
  • Pengisian sekali per periode (prevention double submit)
  • Hasil disimpan secara anonim
  • Link publik via token (tanpa login) untuk distribusi ke semua karyawan

⚙️ Kelola Survey (Admin/HR)

  • Buat dan edit dimensi survey
  • Tambah pertanyaan per dimensi (incl. reverse-scored)
  • Kelola periode: buat, edit, aktifkan, hapus
  • Hanya 1 periode bisa aktif sekaligus
  • Salin link publik survey untuk distribusi

🏷️ Interpretasi Skor

  • ≥ 75 → Engaged
  • 55–74 → Moderate
  • < 55 → Disengaged
  • Skor dinormalisasi dari rata-rata Likert 1–5
  • Reverse-scored questions dihitung terbalik otomatis

19. Training Needs Analysis (TNA)

/people/tna
HR

Sistem asesmen kompetensi karyawan berbasis multi-rater (self, atasan, rekan) dengan pembobotan yang dapat dikonfigurasi per periode.

Alur TNA

  1. HR buat Periode TNA (nama, tahun, tanggal, bobot)
  2. HR tambahkan karyawan yang akan diases
  3. HR tambahkan assessor (atasan & rekan) per karyawan
  4. Sistem generate link token unik per assessor
  5. Assessor isi form kompetensi via link (tanpa login)
  6. HR tutup periode → sistem hitung hasil weighted average
  7. Lihat hasil: gap kompetensi & rekomendasi pelatihan

Konfigurasi Bobot Penilaian

Sumber PenilaianDefault Bobot
Self Assessment20%
Atasan Langsung50%
Rekan Kerja30%
Bobot dapat diubah per periode sesuai kebutuhan organisasi.

Hasil TNA per Karyawan

  • Matrix: kompetensi × [self, atasan, rekan, overall]
  • Target kompetensi per jabatan/dept
  • Gap = overall − target → kompetensi di bawah target → masuk daftar kebutuhan training

22. Pertanyaan Umum (FAQ)

Kenapa status event tidak berubah ke COMPLETED?

  • Pastikan 6 modul wajib sudah di-mark selesai: Content, Loyalty, VM, Creative, Exhibitors, Sponsors
  • Klik tombol "Tandai Selesai" di masing-masing modul setelah semua data diisi
  • Status COMPLETED baru muncul setelah semua 6 modul selesai

Form modul jadi read-only, bagaimana membukanya?

  • Form terkunci karena modul sudah di-mark selesai
  • Hanya Admin yang bisa membuka kembali
  • Admin: hapus record di tabel event_completions untuk modul tersebut

Berita di dashboard tidak muncul atau lambat?

  • Berita dimuat via AJAX setelah halaman tampil — tunggu beberapa detik
  • Jika gagal total, kemungkinan server tidak bisa mengakses internet
  • Berita di-cache hingga tengah malam — fresh setiap pagi
  • Cache bisa dihapus manual oleh Admin

Traffic event tidak sesuai dengan yang diinput?

  • Traffic dihitung berdasarkan rentang tanggal event (start_date s.d. end_date)
  • Pastikan tanggal input traffic sesuai dengan tanggal event
  • Jika 2 event berlangsung bersamaan di mall berbeda, traffic terhitung di keduanya

Harga BBM tidak update otomatis?

  • Harga BBM dikelola manual (sumber tidak bisa di-scrape otomatis)
  • Admin klik ikon pensil di kartu BBM dashboard untuk update
  • Link "Cek terbaru" membuka situs resmi Pertamina Patra Niaga

Link survey EEI untuk karyawan yang tidak punya akun?

  • Di menu /people/eei/manage, setiap periode punya survey token
  • URL publik: /eei/{token} — bisa dibagikan tanpa login
  • Double submit dicegah via cookie browser

Bagaimana import traffic dari Excel?

  • Buka /traffic/import
  • Upload file Excel — sistem deteksi tanggal & format pintu otomatis
  • Preview hasil parsing sebelum simpan
  • Bisa upload banyak file sekaligus (multi-upload)

Apakah data bisa diekspor ke Excel / PDF?

  • Technical Meeting Summary: ada tampilan print-friendly (Ctrl+P)
  • Export Excel native belum tersedia di v1.3
