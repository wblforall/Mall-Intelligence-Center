👥 Langkah 3: Tambah Assessor (Atasan & Rekan)

Assessor Atasan

Klik + Atasan di card karyawan. Sistem otomatis menampilkan rantai atasan dari data master:

  • Atasan langsung karyawan ditampilkan pertama
  • Jika atasan tidak aktif / posisi kosong, sistem lanjut ke level atas
  • Rantai terus sampai tidak ada lagi atasan atau mencapai level G1
  • Centang atasan yang akan menjadi assessor → Simpan

Assessor Rekan

Klik + Rekan di card karyawan. Picker menampilkan semua karyawan aktif:

  • Karyawan se-departemen ditampilkan di bagian atas
  • Karyawan lintas dept di bagian bawah
  • Ketik nama untuk filter cepat
  • Centang satu atau lebih rekan → Simpan
Rantai Atasan Otomatis: Sistem menelusuri rantai atasan berdasarkan field atasan_id di data karyawan. Jika ada posisi kosong (atasan tidak aktif), sistem tetap menelusuri ke atasnya lagi — sehingga tidak ada rantai yang putus.

📊 Langkah 6: Melihat & Membaca Hasil Gap Analysis

Di halaman periode, klik tombol "Lihat Hasil" (atau ikon grafik) di card karyawan
Tabel gap analysis tampil per kompetensi. Tersedia kolom: Target, Self, Atasan, Rekan, Avg Tertimbang, Gap, Status
Bagian bawah menampilkan Rekomendasi Training untuk semua kompetensi dengan gap positif (di bawah target)
Klik Print untuk cetak laporan

🧮 Rumus Perhitungan Nilai Akhir

Avg = (Self × Wself + Atasan × Watasan + Rekan × Wrekan) ÷ ΣW
Hanya assessor yang sudah Submit yang masuk perhitungan. Bobot assessor yang belum submit tidak dihitung.
Contoh: Bobot Self=20, Atasan=50, Rekan=30. Atasan belum submit. Maka:
Avg = (Self × 20 + Rekan × 30) ÷ (20 + 30) = (Self × 20 + Rekan × 30) ÷ 50

🎯 Membaca Hasil Gap

Gap = Target − Avg. Nilai positif berarti karyawan masih di bawah target; nilai negatif berarti melampaui.

GapStatusArtiTindakan yang Disarankan
+1.01 ke atas Perlu Perhatian Jauh di bawah target — lebih dari 1 level Prioritas training / coaching intensif
+0.01 – +1.00 Di Bawah Target Di bawah target, selisih kecil Cek rekomendasi training / mentoring
0 Sesuai Target Tepat di level target Pertahankan — monitor berkala
−0.01 ke bawah Melampaui Target Di atas target Kandidat mentor atau promosi jabatan

🎓 Rekomendasi Training Otomatis

Bagaimana Rekomendasi Muncul?
Sistem mengidentifikasi semua kompetensi dengan gap > 0
Untuk setiap kompetensi tersebut, query program training yang ter-link ke kompetensi itu
Filter: hanya program berstatus Upcoming atau Ongoing
Ditampilkan di bagian bawah hasil TNA, dikelompokkan per kompetensi
Syarat agar muncul:
  1. Program training sudah di-link ke kompetensi yang relevan (di halaman detail program)
  2. Status program harus Upcoming atau Ongoing

🔒 Menutup / Membuka Kembali Periode

Periode dapat ditutup setelah semua assessor submit. Klik Tutup Periode di header halaman periode.

  • Status Closed: tidak bisa tambah karyawan atau assessor baru
  • Status Open: masih bisa dimodifikasi
  • Untuk membuka kembali, klik tombol yang sama (Buka Kembali)
🗑️ Hapus Karyawan dari Periode

Klik ikon hapus di card karyawan. Sistem akan menghapus semua data assessment (self, atasan, rekan) beserta item penilaiannya dalam satu transaksi.

Modul 5

EEI Survey — Employee Engagement Index

Mengukur tingkat keterlibatan dan kepuasan karyawan secara periodik. Hasilnya diagregat per departemen, per level jabatan, dan per dimensi engagement.

💡 Konsep EEI

EEI mengukur seberapa terlibat dan puas karyawan terhadap pekerjaan dan lingkungan kerja mereka. Survei diisi secara anonim oleh karyawan yang login ke sistem.

  • Tersusun dari beberapa Dimensi (mis: Keterlibatan Kerja, Pengembangan Karir, dll)
  • Setiap dimensi berisi beberapa Pertanyaan dengan skor Likert 1–5
  • Hasil dikumpulkan per periode, bisa dibanding antar periode
  • Bisa ada reversed question (pertanyaan negatif, skor dibalik)

🔧 Troubleshooting Umum

Masalah Penyebab Solusi
Semua kompetensi tampil di TNA, bukan yang sudah dipilih Dept atau jabatan belum punya assignment kompetensi Lakukan Flow C (assign ke dept) atau Flow D (assign ke jabatan) di Competency Framework
Gap tidak tampil di hasil TNA Target level belum di-set untuk dept/jabatan tersebut Buka Tab Pemetaan Target → set angka target → Simpan
Rekomendasi training tidak muncul Program belum di-link ke kompetensi, atau statusnya bukan Upcoming/Ongoing Buka detail program training → centang kompetensi → Simpan. Pastikan status = Upcoming/Ongoing
Assessor tidak bisa buka link form Token sudah direset / link salah Klik Regenerate Token di baris assessor → salin link baru → kirim ulang
Rantai atasan tidak tampil / kosong Field atasan_id di data karyawan kosong atau semua atasan tidak aktif Cek data karyawan → pastikan atasan_id terisi dan ada setidaknya satu atasan aktif di rantai
Tidak bisa tambah karyawan/assessor ke TNA Periode berstatus Closed Klik tombol "Buka Kembali" di header halaman periode
EEI Survey tidak muncul untuk karyawan Tidak ada periode aktif People → EEI → Kelola → Aktifkan periode yang diinginkan
Realisasi budget training = 0 Status kehadiran peserta belum diupdate ke "Hadir" Detail program → edit peserta → set Status Kehadiran = Hadir
