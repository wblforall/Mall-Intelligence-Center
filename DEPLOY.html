🚀 Panduan Deploy — Mall Intelligence Center

Prosedur transfer kode via USB + update database ke server lokal tujuan

⚠️ Sebelum Mulai

  • Data di server tujuan TIDAK akan hilang — Git hanya mengubah kode, bukan database.
  • Folder public/uploads/ tidak disentuh sama sekali.
  • File .env di server tujuan tidak akan tertimpa.
  • Migration hanya menambah kolom/tabel, tidak menghapus data.

A Di Mesin Sumber (XAMPP Development)

1
Commit semua perubahan terbaru
Buka Command Prompt atau Terminal, arahkan ke folder proyek:
cd C:\xampp\htdocs\mall-intelligence-center
Cek apakah ada perubahan yang belum di-commit:
git status
Kalau ada file yang berubah, commit sekarang:
git add . git commit -m "Update sebelum deploy ke production"
Kalau output git status sudah "nothing to commit", lewati langkah commit.
2
Export database
Buka Command Prompt, jalankan:
C:\xampp\mysql\bin\mysqldump -u root nama_database > C:\export_mic.sql
Ganti nama_database dengan nama DB yang ada di .env (cek nilai database.default.database).
3
Copy ke USB
Salin dua hal ini ke USB:
  • Seluruh folder mall-intelligence-center/
  • File C:\export_mic.sql
Yang TIDAK perlu ikut disalin (boleh dihapus dari USB untuk menghemat ruang): vendor/, writable/cache/, writable/session/

B Di Server Tujuan — Jika Belum Ada Kode Sama Sekali

Ikuti bagian ini hanya jika server tujuan belum pernah diisi kode MIC sebelumnya.
1
Tancapkan USB, copy folder ke htdocs
xcopy /E /I E:\mall-intelligence-center C:\xampp\htdocs\mall-intelligence-center
Sesuaikan huruf drive USB (E:) jika berbeda.
2
Install dependencies
cd C:\xampp\htdocs\mall-intelligence-center php composer.phar install --no-dev
3
Buat file .env
copy env .env
Edit file .env, sesuaikan:
CI_ENVIRONMENT = production app.baseURL = 'http://localhost/mall-intelligence-center/public/' database.default.hostname = localhost database.default.database = nama_database database.default.username = root database.default.password = database.default.DBDriver = MySQLi
4
Import database
Buat database dulu di phpMyAdmin, lalu import:
C:\xampp\mysql\bin\mysql -u root nama_database < E:\export_mic.sql
5
Jalankan migration
php spark migrate
Untuk memastikan semua migration sudah jalan, terutama jika SQL export lebih lama dari kode.

C Di Server Tujuan — Update Rutin (Sudah Ada Kode Sebelumnya)

Ini yang dilakukan setiap kali ada update dari development.
1
Tancapkan USB
Pastikan folder mall-intelligence-center/ dari USB sudah berisi commit terbaru (sudah dicommit di langkah A).
2
Pull perubahan dari USB menggunakan Git
Buka Command Prompt di server tujuan, arahkan ke folder proyek:
cd C:\xampp\htdocs\mall-intelligence-center
Tambahkan USB sebagai remote (hanya sekali, pertama kali saja):
git remote add usb E:\mall-intelligence-center
Setiap update berikutnya, cukup jalankan:
git pull usb main
Ganti E: dengan huruf drive USB yang aktif. Kalau drive berubah, update dengan: git remote set-url usb F:\mall-intelligence-center
3
Jalankan migration (jika ada)
php spark migrate
Aman dijalankan meski tidak ada migration baru — tidak akan mengubah data.
4
Clear cache (opsional)
php spark cache:clear

D Kalau Ada File Upload Baru yang Perlu Dipindah

File upload user (foto dokumentasi, bukti tanda terima, dll.) tidak ditrack Git. Kalau ada file baru di development yang perlu ada di production, salin manual:
# Copy folder uploads dari development ke USB xcopy /E /I C:\xampp\htdocs\mall-intelligence-center\public\uploads E:\uploads_mic # Di server tujuan, merge ke folder uploads production xcopy /E E:\uploads_mic C:\xampp\htdocs\mall-intelligence-center\public\uploads
Gunakan flag /Y jika ingin skip konfirmasi overwrite.

✅ Ringkasan Perintah Update Rutin

Di server tujuan — setiap ada update
cd C:\xampp\htdocs\mall-intelligence-center git pull usb main php spark migrate
Tiga baris ini sudah cukup untuk update rutin.
